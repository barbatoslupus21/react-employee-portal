# 1. System Overview

## What is RepConnect?

RepConnect is an integrated Human Resource Management System (HRMS) designed to manage the full employee lifecycle — from daily time logging and leave filing to payroll, training, performance evaluation, and IT support. The system is web-based and accessible through any modern browser.

## User Roles & Permissions

The system has permission-based access. Your role determines which modules and actions you can see.

| Role | Description |
|------|-------------|
| **Employee** | Standard access — file leave, view payslip, submit forms |
| **Supervisor / Manager** | Approve leaves, evaluate team members |
| **HR** | Manage leave configurations, employees, evaluations |
| **HR Manager** | Senior HR access with additional approval authority |
| **Clinic** | Review and approve leave from a health/wellness standpoint |
| **IAD** | Internal audit review step in leave workflows |
| **Accounting** | Access financial records |
| **MIS / IT** | Manage and resolve IT support tickets |
| **Admin** | Full system access — configuration, uploads, approvals |
| **News** | Create and publish announcements |

## Logging In

1. Open the RepConnect application in your browser.
2. Enter your **Employee ID Number** as your username.
3. Enter your **password**.
4. Click **Login**.

> **First-time login:** If your account has been flagged for a password change, you will be redirected to set a new password before accessing the system.

> **Account lockout:** After multiple failed login attempts, your account will be temporarily locked. Contact the system administrator to unlock it.

---

## Notifications

The bell icon in the top navigation bar shows your in-app notifications. You will receive notifications for:

- Leave request status changes (approved/disapproved)
- PRF status updates
- New certificates uploaded for you
- Training assignments and evaluation requests
- MIS ticket status updates
- Finance events (payslip uploaded, loan updates)

Click a notification to go directly to the related record.

---

## Logging Out

Click your profile avatar or name in the top-right corner, then select **Logout**.

---

## Password Management

Navigate to **Profile Settings > Change Password** to update your password at any time. You must enter your current password before setting a new one.
