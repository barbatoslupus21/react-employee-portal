# 13. System Feedback & What's New

## Overview

The Feedback module lets employees rate their experience with the RepConnect system. The What's New feature keeps everyone informed about system updates and new features.

---

## How to Submit System Feedback

### When the Feedback Modal Appears
A feedback popup will periodically appear after logging in. You can:
1. Select a **star rating** (1–5):
   - 1 star: Very Unsatisfied
   - 2 stars: Unsatisfied
   - 3 stars: Neutral
   - 4 stars: Satisfied
   - 5 stars: Very Satisfied
2. Optionally, type a comment about your experience.
3. Click **Submit**.

> You can only submit one feedback per calendar month. If you have already submitted feedback this month, the modal will not ask again.

### Submitting Feedback Manually
1. Navigate to **Feedback** in the sidebar (if available).
2. Rate the system and add your comments.
3. Click **Submit**.

---

## What's New (System Updates)

When there are new features or improvements, a **What's New** notification will appear.

1. Click the notification or the **What's New** banner.
2. Read the update notes.
3. Click **Got It** or **Close** to dismiss.

Update notes include:
- Version number (e.g., v1.2.0)
- List of new features
- Bug fixes and improvements

---

## Feedback Administration (Admin)

Administrators can:
- **Enable/Disable** the feedback modal globally
- **View all feedback** submitted by employees, including ratings and comments
- **View feedback trends** over time

> Feedback is reviewed to improve the system and prioritize new features.
