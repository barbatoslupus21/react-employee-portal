# 10. Calendar & Timelogs

## Overview

The Calendar module combines a company-wide event calendar with an employee time-in/time-out logging system. Employees can see scheduled events, holidays, meetings, and deadlines, as well as log their attendance.

---

## Calendar Events

### Event Types

| Type | Description |
|------|-------------|
| **Important** | High-priority company notices |
| **Meeting** | Scheduled team or company meetings |
| **Task** | Task deadlines and assignments |
| **Reminder** | Personal or shared reminders |
| **Deadline** | Critical due dates |
| **Legal/Special Holiday** | Government-declared holidays |
| **Day Off** | Company-declared non-working days |
| **Company Holiday** | Internal company holidays |

### Event Repetition Options

| Option | Description |
|--------|-------------|
| Once | One-time event |
| Daily | Repeats every day |
| Weekly | Repeats on the same day each week |
| Monthly | Repeats on the same date each month |
| Yearly | Repeats annually |

---

## How to View the Calendar

1. Navigate to **Calendar** in the sidebar.
2. The calendar shows the current month by default.
3. Click the **<** and **>** arrows to navigate between months.
4. Click on a date to see events for that day.
5. Click on an event to view its details.

---

## How to Create an Event (Admin/HR)

### Step 1 — Click on a Date
Click on the target date in the calendar or click the **Create Event** button.

### Step 2 — Fill the Event Form

| Field | Description |
|-------|-------------|
| **Title** | Event name |
| **Type** | Select from event types listed above |
| **Date / Date Range** | Start and end date/time |
| **Repetition** | How often it repeats |
| **Description** | Additional details |
| **Scope** | All employees or specific selected members |

### Step 3 — Save
Click **Save**. The event appears on the calendar for all selected participants.

---

## How to Log Your Time (Timelogs)

### Time-In
1. Navigate to **Calendar** and locate the Timelog section, or use the time-in button on the dashboard.
2. Click **Time In** at the start of your work day.
3. Your time-in is recorded with a timestamp.

### Time-Out
1. At the end of your work day, click **Time Out**.
2. Your time-out is recorded.

> Timelogs are managed and reviewed by HR and administrators for attendance monitoring.

---

## Event Sharing

When an event has a **specific members** scope, only selected employees can see that event on their calendar. Events set to **All** are visible company-wide.

Participants receive a notification when a new event is shared with them.

---

## Event View Tracking

The system tracks whether each participant has **viewed** a calendar event. Administrators can see who has and has not viewed a specific event.
